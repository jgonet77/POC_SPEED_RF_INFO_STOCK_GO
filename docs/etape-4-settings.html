⚙️ Étape 4 : Settings & Configuration Dynamique

Gestion des paramètres API et création SettingsActivity (~2 heures)

Objectif

Permettre à l'utilisateur de modifier :

• IP du serveur API (défaut: 192.168.1.20)
• Port du serveur API (défaut: 8000)
• Persister les paramètres en SharedPreferences
• Relancer le test de connexion avec les nouveaux paramètres

1. ConfigManager : Gestion des Préférences

class ConfigManager(context: Context) { private val sharedPrefs = context .getSharedPreferences("api_config", Context.MODE_PRIVATE) fun getApiHost(): String = sharedPrefs .getString("api_host", "192.168.1.20") ?: "192.168.1.20" fun getApiPort(): String = sharedPrefs .getString("api_port", "8000") ?: "8000" fun setApiHost(host: String) { sharedPrefs.edit() .putString("api_host", host) .apply() } fun setApiPort(port: String) { sharedPrefs.edit() .putString("api_port", port) .apply() } }

Avantages

SharedPreferences : Stockage persistant simple

Singleton Pattern : Une seule instance par app

Default Values : Fallback si clé manquante

apply() : Asynchrone, non-bloquant

2. SettingsActivity

class SettingsActivity : AppCompatActivity() { private lateinit var configManager: ConfigManager private lateinit var hostInput: EditText private lateinit var portInput: EditText private lateinit var retryButton: Button override fun onCreate( savedInstanceState: Bundle? ) { super.onCreate(savedInstanceState) setContentView(R.layout.activity_settings) configManager = ConfigManager(this) hostInput = findViewById(R.id.hostInput) portInput = findViewById(R.id.portInput) retryButton = findViewById(R.id.retryButton) // Charger les valeurs actuelles hostInput.setText( configManager.getApiHost() ) portInput.setText( configManager.getApiPort() ) retryButton.setOnClickListener { val newHost = hostInput.text.toString() val newPort = portInput.text.toString() configManager.setApiHost(newHost) configManager.setApiPort(newPort) AppLogger.log( "Settings: Changed API to " + "$newHost:$newPort" ) Toast.makeText( this, "Configuration mise à jour", Toast.LENGTH_SHORT ).show() finish() } } }

4. Intégration dans ApiClient

fun init(context: Context) { val config = ConfigManager(context) val host = config.getApiHost() val port = config.getApiPort() val retrofit = Retrofit.Builder() .baseUrl("http://$host:$port/") .addConverterFactory( GsonConverterFactory.create() ) .build() apiService = retrofit .create(StockApiService::class.java) }
